# Subir o projeto com GitHub Desktop (sem usar terminal)

Se o comando `git` não funciona no terminal, use o **GitHub Desktop**.

## 1. Instalar o GitHub Desktop

- Baixe: https://desktop.github.com/
- Instale e abra o programa.
- Faça login na sua conta do GitHub (ou crie uma).

## 2. Publicar este projeto no GitHub

1. No GitHub Desktop: **File** → **Add local repository**.
2. Clique em **Choose...** e selecione a pasta:
   `C:\Users\Pichau\Documents\Project Fin\pricing-site`
3. Se aparecer "This directory does not appear to be a Git repository":
   - Clique em **create a repository** (ou "Create Repository").
   - Deixe o nome como `pricing-site` e **Create Repository**.
4. Você verá a lista de arquivos alterados. Na caixa de texto embaixo de "Summary", escreva por exemplo: **Primeiro commit**.
5. Clique em **Commit to main**.
6. No topo: **Publish repository**.
7. Escolha o nome (ex.: `pricing-site`), deixe **Public** e clique em **Publish Repository**.

Pronto. O projeto estará no GitHub e você pode conectar no Render.
